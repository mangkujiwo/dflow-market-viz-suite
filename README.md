## 🛠️ Installation & Setup Guide

1. **Clone the repository**  
2. **Configure your `.env` file** for APIs (Kalshi, OpenAI, Anthropic, or other connectors)
3. **Install dependencies** using `pip install -r requirements.txt` or `npm install`
4. **Run locally** with `python manage.py runserver` or `npm start`
5. **Launch the web UI**—unveil live markets on your local browser!

## 🧑‍💻 Example Profile Configuration

Paste this in `~/.eventsight/config.yaml`:

    profile:
      user_email: "your@email.com"
      language: "en"
      preferred_markets:
        - "Kalshi: US Elections"
        - "Sportsbook: Premier League"
      notifications:
        email: true
        push: true
        ai_summaries: true
      ai_providers:
        - "OpenAI"
        - "Anthropic Claude"
      custom_alerts:
        - event: "Kalshi > Inflation Rate > Crosses 3%"
          action: "Send Email & Summarize via OpenAI"

---

## 🛎️ Example Console Invocation

To launch with your profile and view US Election market data with AI summaries:

    $ eventsight --market "Kalshi: US Elections" --profile "~/.eventsight/config.yaml"
